#  React Todo App

A Todo Application built using React and Material UI with LocalStorage support.

---

##  Tech Stack

- React.js
- Material UI (MUI)
- Context API
- Custom Hooks
- UUID
- LocalStorage

---

##  Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Persistent storage using LocalStorage
- Clean and responsive UI
